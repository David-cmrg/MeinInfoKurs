Hello World

📄 HelloWorld.java · Version 1.0 · 25.02.2024 · ⭐ Einsteiger
AUFGABENSTELLUNG

Die Aufgabe der heutigen Stunde ist Hello World! Hello World ist ein Standardprogramm, welches man in jeder Programmiersprache als erstes schreibt.

Schreibe die folgende Zeile in den Haupt-Abschnitt:

System.out.println("Hello World");

Lösung

HelloWorld.java 
1/**
2 * Das fertige Java Programm "HelloWorld.java"
3 *
4 * @version 1.0 vom 25.02.2024
5 * @MeinInfoKurs.github.io
6 */
7
8public class HelloWorld {
9
10    public static void main(String[] args) {
11        System.out.println("Hello World");
12    }
13}

Erklärung

In dieser Aufgabe wird lediglich die Ausgabe eines Strings (String = Text) gefordert. Das ist der einfachste Schritt in Java!

In BlueJ kannst du mit ⌃ STRG + U automatisch einen System.out.println()-Befehl erzeugen. Schreibe dann "Hello World" in die Klammern.

⚠️ Vergiss das Semikolon am Ende nicht — ohne es wird dein Programm nicht kompilieren!

System.out.println("Hello World");
